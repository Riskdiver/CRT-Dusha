## [1.2] - 2026
### Added
- Frame step debug mode: locks decay cycle to discrete steps for frame-by-frame inspection
- Frame counter overlay: displays current frame number in the top right corner during debug mode
- Phosphor Tint: color cast option for monochrome phosphor simulation (P31 green, P3 amber, etc.)

### Changed
- Voodoo2 16-bit quantization corrected to better represent original RGB565 hardware behavior
- Default preset and several uniform default values updated for better out-of-the-box experience

### Fixed
- Multiple tooltip corrections across all parameter categories

### Removed
- Redundant code cleanup
